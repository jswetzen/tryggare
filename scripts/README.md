# Scripts

Helper scripts used during development. None of these are required at runtime.

| Script | Purpose |
|---|---|
| `validate-docker-setup.sh` | Quick sanity check for Docker / Podman + Compose installation before running the dev environment. |
| `take_screenshots.py` | Selenium script that logs into a running dev environment (localhost:5173) and captures the screenshots used in the project README. Useful after a significant UI change to refresh `docs/screenshots/`. |
