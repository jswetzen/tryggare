## Project Organization

### Documentation
All notes, implementation documentation, design documents, and other markdown files should be saved in the `/docs` folder.

### Tools and Scripts
Any ad-hoc scripts, utilities, or tools that are not part of the main application should be saved in the `/tools` folder.

## Directory Structure

```
/docs           - All documentation, notes, and implementation files
/tools          - Ad-hoc scripts and utilities not part of the main app
```
